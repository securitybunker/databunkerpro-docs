This section collects short, task-focused guides for operations you perform against a running Databunker Pro instance.

## Available how-tos

- [Update the license key](/pro/howtos/update-license) — apply a new license to a running instance via the API or the web interface.
- [Clean the users table](/pro/howtos/clean-users-table) — remove all user records and related data for a full reset (testing, staging, or decommissioning).

<Note>
These guides assume you already have Databunker Pro [installed](/pro/installation/docker-compose) and an admin (root) access token. See [Generate admin credentials](/pro/installation/generate-admin-credentials) if you have not completed the initial setup.
</Note>
